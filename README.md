# My Emacs stuff
- I did this emacs config after trying both the spacemacs and DOOM Emacs distros (for less-than 20 minues). I guess I like my shii made from scratch.
- Fyi it does not have any evil (or other third-party) keybinds, just pure Emacs holiness. If you think it is a bit unergonomic (or some shii) just remap CTRL to Caps Lock and if you have Esc bound to Caps Lock... Then dont use emacs (or use evil-mode). At this point in time (27/10/2026) i have not implemented any note-taking setup but i will try to add it (hopefully not Org-mode).
